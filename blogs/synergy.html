synergy is the new decentralized, opensource, free, fast and secure VPN protocol.

how will it work?

basically, its more like TOR + TORRENT

user will be client as well as the server. when user connects to synergy, it will act as a client who can connect to nodes and can surf internet normally, as well as the server, who will act as a node for other users.

synergy first filter out nodes that have good bandwidth and prioritize them more. other nodes that have low bandwidth at certain scale will not be considered as node that means, user with very slow bandwidth will only be a client, not a server.

synergy georgraphically checks for the fastest route to the server user wants to connect. for example: if the user is in india, tries to connect to russian site. it will only choose the node that are near to the displacement of both the locations. there is no point of connecting to the server that are too far from the displacement. this technique will eventually make internet speed way more faster.

other thing that will make synergy faster is that it will only connect from 1 to maximum of 5 nodes at a time which will keep the number of jumps low. the more people use it, the more nodes will be there, which eventually make it more faster.

synergy also keeps on changing its nodes after certain interval of time which keeps the user untrackable. that means IP will no more be your identity.
